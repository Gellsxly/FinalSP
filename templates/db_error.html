Database MySQL Tidak Aktif

Aplikasi tidak dapat terhubung ke database MySQL di localhost Anda. Silakan ikuti langkah-langkah di bawah ini untuk mengaktifkannya:

  1. Buka aplikasi XAMPP Control Panel di PC Anda.
  2. Temukan baris modul MySQL.
  3. Klik tombol Start di samping MySQL hingga indikatornya berubah menjadi hijau.
  4. Segera refresh halaman ini setelah MySQL aktif.
